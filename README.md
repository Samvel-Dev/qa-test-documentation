# QA Test Documentation Portfolio

This repository demonstrates structured QA documentation created as part of my practical training as a Junior QA Engineer.

Contents:

01 – Test Plan  
Defines testing scope, objectives and environment

02 – Test Cases  
Sample functional test cases for login module

03 – Bug Reports  
Example defect reports with severity levels and reproduction steps

04 – Test Strategy  
Overview of testing approach and tools used

Tools and technologies:

- Manual testing techniques
- Selenium WebDriver (basic automation)
- Postman (API testing)
- Jira (bug tracking)

Purpose:

To demonstrate practical software testing workflow and documentation skills used in real QA environments.
